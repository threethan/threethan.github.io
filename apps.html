Install Guide

Progressive web apps can be installed through your web browser.
The process is a bit different depending on your device.

iOS Safari (iPhone, iPad)

  1. Open the Web App in Safari

    2. -> If you're not already using Safari, open it from your homescreen and navigate to:
    threethan.github.io/apps.html
  2. Tap the icon on the bottom bar
  3. Select "Add to Home Screen"

Chrome (Windows, Mac, ChromeBook)

This does not work on iOS
  1. Open the Web App in Chrome
  2. Click the icon on the right of the URL bar

Chrome (Android)

I recommend you install through Google Play  instead
  1. Open the Web App in Chrome
  2. Tap the menu button to the right of the URL bar
  3. Tap "Install"
